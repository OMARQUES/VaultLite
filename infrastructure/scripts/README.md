# Infrastructure scripts

Repeatable infrastructure and local-development scripts live here.

Current scripts:
- `local-issue-invite.mjs`: issue a local invite against the dev API
- `reset-local-state.mjs`: clear local wrangler state for D1/R2 dev storage
- `dev-local.mjs`: run API and web dev servers in a single terminal
- `validate-migrations.mjs`: validate local migration order and convergence
- `smoke-local-flow.mjs`: run the local auth smoke flow
